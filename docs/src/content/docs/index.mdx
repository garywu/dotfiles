## Structure

### System Configuration
- Nix and Home Manager for declarative package management
- Chezmoi for machine-specific secrets and templates
- Homebrew for macOS GUI applications

### Development Tools
- Modern CLI replacements with better performance and usability
- Language-specific toolchains (Python, Node.js, Go, Rust)
- Cloud provider CLIs (AWS, GCP, Cloudflare)

### Workflow Automation
- Git hooks and pre-commit configurations
- GitHub Actions for CI/CD
- Session tracking and management scripts

## Key Components

- **Package Management**: Reproducible development environments with Nix
- **Shell Configuration**: Fish shell with custom functions and completions
- **Security**: Secret management patterns and Git privacy protection
- **Testing**: Comprehensive test suites for configurations and scripts
- **Documentation**: Maintained with Starlight and GitHub Pages
